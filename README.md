# 📦 Logic Box - Pattern Generator & Number Analyzer

## 📖 Description
Logic Box is a Python-based menu-driven program that allows users to generate different star patterns and analyze numbers (even/odd). It is a great beginner project to understand loops, conditions, and logic building.

---

## 🚀 Features

### 🔷 Pattern Generator
- Right Angle Triangle
- Reverse Right Angle Triangle
- Left Angle Triangle
- Reverse Left Angle Triangle
- Pyramid Shape
- Reverse Pyramid Shape
- Diamond Shape

### 🔢 Number Analyzer
- Check numbers between a range
- Identify even and odd numbers

### 🔁 Menu System
- Interactive menu using `while` loop
- Continuous execution until user exits

---

## 🧑‍💻 Technologies Used
- Python
  - Loops (`for`, `while`)
  - Conditional Statements (`if-elif-else`)
  - User Input (`input()`)
  - Basic Logic Building
